Accessibility Statement

I'm an accessibility tester, so this site is held to the same standard I hold the sites I audit. It should work for everyone — with a keyboard, with a screen reader, at high zoom, and in any browser.

Conformance status

As of July 2026, this site is designed to meet WCAG 2.2 Level AA.

How this site was tested

I tested this site manually, using the same core approach I use in my audit work:

  • Screen reader testing with NVDA in Firefox on Windows 11
  • Keyboard-only navigation, including focus visibility and skip-link behavior
  • Accessible names and roles inspected with ANDI
  • Color contrast verified with the Colour Contrast Analyser (all text meets or exceeds 4.5:1)
  • Structure review: landmarks, heading hierarchy, and text alternatives for all meaningful images
  • Reflow and zoom checked up to 400%
